OTOD ยืนยันแล้ว + มี SN จากชีตเก่าด้วย ไม่ต้องตามซ้ำ
🟡
ได้ตู้แล้ว รอยืนยันเอกสาร
-
OTOD ยังไม่อัปเดต แต่มีหลักฐานว่าได้ตู้แล้ว

🟡 ได้ตู้แล้ว รอยืนยันเอกสาร 0 รายการ

▶

คนกลุ่มนี้ OTOD ยังไม่อัปเดตสถานะเป็นติดตั้งแล้วอย่างเป็นทางการ แต่มีหลักฐานว่าได้รับตู้ไปแล้วจริง (จากข้อมูลชุดเก่า หรือเชื่อมต่อแอปแล้ว) — ไม่ต้องเลือกไปติดตั้งซ้ำ แค่รอเอกสารยืนยันจาก OTOD เท่านั้น

ลำดับคำนำหน้าชื่อนามสกุลเลขบัตรประชาชน เบอร์มือถือSN ตู้เลข ARจังหวัดอำเภอตำบลสถานะในระบบ OTODเอกสาร/ภาพ/วีดีโอ

❓ เชื่อมต่อแอปแล้ว แต่ไม่พบข้อมูลในระบบ OTOD 0 รายการ

▶
ลำดับSN ตู้รหัสผู้ใช้แอปชื่อ (จากแอป)นามสกุล (จากแอป) เลขบัตรประชาชน (จากแอป)เบอร์มือถือ (จากแอป)

⛔ OTOD บอกว่าติดตั้งแล้ว แต่ยังไม่เชื่อมต่อแอป 0 รายการ

▶
ลำดับคำนำหน้าชื่อนามสกุลเลขบัตรประชาชน เบอร์มือถือSN ตู้เลข ARจังหวัดอำเภอตำบล

✅ ได้ตู้แล้ว ยืนยันซ้ำจากข้อมูลชุดเก่า 0 รายการ

▶

คนกลุ่มนี้ OTOD ยืนยันติดตั้งเสร็จอย่างเป็นทางการแล้ว (ไม่ใช่ข้อมูลที่ระบบอนุมานเอง) และมีข้อมูลจากชีตติดตามตู้/Serial Board ชุดเก่ายืนยันซ้ำอีกชั้นว่าได้ตู้ (SN) ไหนไปจริง — ยืนยันครบทั้งสองทางแล้ว ไม่ต้องตามให้เชื่อมต่อแอปด่วนเหมือนกลุ่มบน

ลำดับคำนำหน้าชื่อนามสกุลเลขบัตรประชาชน เบอร์มือถือSN ตู้เลข ARจังหวัดอำเภอตำบล

✅ ตรงกันแล้ว (เชื่อมต่อแอป + OTOD อัปเดตแล้ว) 0 รายการ

▶
ลำดับคำนำหน้าชื่อนามสกุลเลขบัตรประชาชน เบอร์มือถือSN ตู้เลข ARจังหวัดอำเภอตำบลสถานะในระบบ OTOD

✍️ กรอกรหัสฐานเอง 0 รายการรอ

ใช้ตอนเกษตรกรได้รับตู้แล้วจริง แต่บัญชีแอปที่เชื่อมมาชื่อไม่ตรงกับคนที่ได้รับ (เช่น ยืมเบอร์คนอื่นเชื่อม) ทำให้ระบบจับคู่อัตโนมัติไม่เจอ — ด้านล่างนี้ระบบจะดึงรายชื่อคนที่น่าจะรอกรอกรหัสฐานขึ้นมาให้อัตโนมัติ (แผนบอกว่าติดตั้งแล้วแต่ระบบยังไม่อัปเดต หรือติดตั้งแล้วแต่ไม่มี SN เลย) หรือจะพิมพ์ค้นหาคนอื่นเพิ่มเติมก็ได้ — พิมพ์รหัสฐาน (เช่น AR_0008) แล้วบันทึก ระบบจะไปเทียบกับชีตรหัสตู้เพื่อดึง SN ตัวจริง แล้วเปลี่ยนสถานะเป็น "ได้ตู้แล้ว" ให้อัตโนมัติทันที

ชื่อ-นามสกุลเลขบัตรประชาชนเบอร์มือถือสถานะปัจจุบันเหตุผลรหัสฐาน
กำลังโหลดรายชื่อ...

📋 รายชื่อที่กรอกรหัสฐานไว้แล้ว 0 รายการ

ชื่อ-นามสกุลเลขบัตรประชาชนเบอร์มือถือรหัสฐานSN ตู้จังหวัดกรอกโดยกรอกเมื่อ
ยังไม่มีรายการที่กรอกรหัสฐานเองเลยครับ

🚫 ทำเครื่องหมายว่าติดตั้งไม่ได้

ใช้เมื่อพบว่าเกษตรกรคนนี้ติดตั้งไม่ได้ด้วยเหตุผลบางอย่าง (เช่น ปั๊มเครื่องยนต์ที่ตู้ไม่รองรับ) — ไม่ว่าจะยังไม่เข้าแผนติดตั้งหรืออยู่ในแผนอยู่แล้วก็ทำเครื่องหมายได้ สถานะนี้แยกอิสระจากสถานะในแผนติดตั้ง แก้ไขเหตุผลหรือยกเลิกเครื่องหมายเมื่อไหร่ก็ได้

ชื่อ-นามสกุลเลขบัตรประชาชนเบอร์มือถือสถานะในระบบ OTOD
พิมพ์ชื่อ/เลขบัตร/เบอร์ด้านบนเพื่อค้นหาคนที่จะทำเครื่องหมายครับ

📋 รายชื่อที่ติดตั้งไม่ได้ตอนนี้ 0 รายการ

ชื่อ-นามสกุลเลขบัตรประชาชนเบอร์มือถือเหตุผลทำเครื่องหมายโดยเมื่อ
ยังไม่มีใครติดสถานะนี้เลยครับ
